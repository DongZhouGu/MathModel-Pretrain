**使用方法**

每个练习文件夹有三个不同类型的文件: 

1.Exercises.ipynb

没有答案代码的文件，这个是你做的练习

2.Solutions.ipynb

运行代码后的结果（不要改动）

3.Exercise_with_Solutions.ipynb

有答案代码和注释的文件

你可以在Exercises.ipynb里输入代码，看看运行结果是否和Solutions.ipynb里面的内容一致，如果真的完成不了再看下Exercise_with_Solutions.ipynb的答案。
